1. STUDY DESIGN OVERVIEW
========================================================================

Model system: HOTAIR RNA pull-down binding proteins
Genes analyzed: 104 (30 candidates + 44 non-candidates + 30 T2DM controls)
Phenotypes: DR (Diabetic Retinopathy), DN (Diabetic Nephropathy),
            DPN (Diabetic Peripheral Neuropathy)
eQTL sources: GTEx v8 (multi-tissue MASHR) vs eQTLGen (whole blood meta-analysis)
GWAS source: FinnGen R13

Key analyses:
  (A) S-PrediXcan TWAS with GTEx v8 weights (baseline)
  (B) S-PrediXcan TWAS with eQTLGen weights (sensitivity)
  (C) Direction consistency & Spearman correlation (A vs B)
  (D) FDR enrichment analysis by group × phenotype × eQTL source
  (E) Mahalanobis covariate matching + post-matching enrichment
  (F) Pull-down vs literature stratification
  (G) Cross-population replication (UK Biobank)

========================================================================
4. ANALYSIS WORKFLOW
========================================================================

Step 1: S-PrediXcan with GTEx v8 weights
  -> Multiple tissues per gene (Nerve_Tibial + Whole_Blood)
  -> Stouffer integration across tissues
  -> ACAT-O sensitivity validation
  -> Output: GTEx TWAS Z-scores per gene × phenotype

Step 2: S-PrediXcan with eQTLGen weights
  -> Single tissue (whole blood)
  -> Same FinnGen R13 GWAS input
  -> Output: eQTLGen TWAS Z-scores per gene × phenotype

Step 3: Direction consistency analysis
  -> Compare Z-score signs between GTEx vs eQTLGen
  -> Binomial sign test
  -> Per group × per phenotype

Step 4: Spearman correlation
  -> Rank-based correlation of GTEx vs eQTLGen Z-scores
  -> Global + per phenotype

Step 5: Enrichment analysis
  -> FDR (BH) correction per analysis domain
  -> Fisher's exact test for group comparison
  -> GTEx baseline -> eQTLGen sensitivity -> Mahalanobis post-matching

Step 6: Mahalanobis matching (R matchit)
  -> Match 30 candidate genes from full-genome pool
  -> Dimensions: log10(Length), GC%, eQTL_SNPs_Mean
  -> Nearest neighbor, Mahalanobis distance, ratio=1

Step 7: Post-matching enrichment validation
  -> Repeat FDR enrichment on matched pairs
  -> SMD diagnostics (Love plot)

Step 8: Stratification by pull-down vs literature
  -> Split 30 candidates by source
  -> Direction consistency per stratum

Step 9: Cross-population replication
  -> UK Biobank independent DR GWAS
  -> Meta-analysis: Cochran's Q, I-squared, 95% PI
  -> External validation: Cai et al. 2026, Sakaue et al. 2021
