RUN 1: eQTLGen Whole_Blood — DR (FinnGen R13)
========================================================================
Date:         2026-06-10 (approximate)
Command:
  python SPrediXcan.py \
    --model_db_path eQTLGen_Whole_Blood.db \
    --covariance 1000G_EUR_covariance.txt.gz \
    --gwas_file FinnGen_R13_DR.sumstats.gz \
    --snp_column rsid \
    --effect_allele_column alt \
    --non_effect_allele_column ref \
    --beta_column beta \
    --se_column se \
    --pvalue_column pval \
    --output_file results/eqtlgen_Whole_Blood_DR.csv \
    --additional_output

Input files:
  - eQTLGen_Whole_Blood.db           (processed weights from eQTLGen Z-scores)
  - 1000G_EUR_covariance.txt.gz
  - FinnGen_R13_DR.sumstats.gz

Model stats:
  - Genes with eQTL: ~16,000 (eQTLGen)
  - SNPs per gene: median ~625 (range 1-5000+)
  - Distinct from GTEx: eQTLGen uses single-SNP meta-analysis Z-scores
    (not MASHR Bayesian shrinkage), resulting in many more SNPs per gene

Output file: results/eqtlgen_Whole_Blood_DR.csv
  - Total rows: ~100-199 (genes tested with full covariance match)
  - n_SNPs_Model: number of SNPs in the prediction model
  - n_SNPs_Matched: number of SNPs matched to GWAS
